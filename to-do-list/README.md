# 오늘의 할일 (To-Do Today)

간단한 하루 단위 할일 관리 SPA입니다. 로컬스토리지에 저장되며 별도 서버가 필요 없습니다.

사용 방법

1. 파일을 브라우저로 열어 사용합니다: `to-do-list/index.html`
2. 제목을 입력하고 `추가` 버튼 또는 Enter로 할 일을 추가합니다.
3. 체크박스로 완료 토글, ✏️로 편집, 🗑️로 삭제합니다.
4. 상단의 필터로 `전체`/`미완료`/`완료`를 전환합니다.

개발자 메모

- 데이터는 날짜별로 분리되어 저장됩니다(키: `todo-today-YYYY-MM-DD`).
- 향후 서버 동기화, 알림, 이력 보기 등을 확장할 수 있습니다.
